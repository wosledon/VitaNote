# VitaNote Design

## UI/UX Design

### Design Principles

1. **Clean & Minimal**: Focus on essential information
2. **Mobile-First**: Responsive design for all screen sizes
3. **Data-Driven**: Visualizations for health trends
4. **Accessible**: WCAG 2.1 AA compliant

### Color Palette

```css
/* Primary Colors */
--primary: #1976d2
--primary-dark: #1565c0
--primary-light: #42a5f5

/* Secondary Colors */
--secondary: #e91e63
--secondary-dark: #c2185b
--secondary-light: #f48fb1

/* Background */
--background: #f5f5f5
--paper: #ffffff

/* Text */
--text-primary: #212121
--text-secondary: #757575
```

### Components

#### Cards

- Date: Rounded corners (12px)
- Color: White with light shadow
- Use: Record display, quick actions

#### Buttons

- Primary: Blue (#1976d2)
- Secondary: Pink (#e91e63)
- Text: Outline style
- Floating Action Button (FAB): Primary with shadow

#### Typography

- H1: 24px, bold
- H2: 20px, bold
- H3: 18px, semibold
- Body: 14px, regular
- Caption: 12px, regular
