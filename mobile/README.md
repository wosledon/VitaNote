# VitaNote Mobile

## Setup

```bash
npm install
npx tauri android init
npx tauri ios init
```

## Development

```bash
# Run on Android
npx tauri android dev

# Run on iOS
npx tauri ios dev
```

## Build

```bash
# Build Android
npx tauri android build

# Build iOS
npx tauri ios build
```

## Features

- Health data recording
- Camera integration
- OCR processing
- AI health assistant
